# Benchmark Generation Plan

## Goal

- HTMLに落とす前に、全13ページの完成イメージ基準を image2 で持つ
- 三幸資料の赤白トーンを踏襲しつつ、現行案より洗練させる

## Generation Order

1. 一覧ボード
2. 表紙
3. 目次
4. 提案の骨子
5. 展開イメージ
6. 今後の進め方
7. 残りページ

## Output Mode

- 1枚の一覧ボード
- 個別スライド画像

## Visual Rules

- 赤白ベース
- 日本のB2B提案資料
- 過剰な広告感は避ける
- 三幸資料より少し洗練
- 文字入りで全体トーンを確認する

## Text Handling Rule

- ベンチマーク段階では文字入り
- 最終HTMLでは文字をHTMLへ戻す
- 画像の文字はそのまま再利用しない

- 表紙の温度感

- 今後の進め方の視線誘導
