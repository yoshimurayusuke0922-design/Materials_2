# Placeholder Plan

## Skill Used

- `deck-placeholder-frame-builder`

## Principle

- 空欄でも「あとで貼る箱」ではなく、完成形を想起できる UI フレームとして見せる
- 各 slide で placeholder composition を変える

## 07 Demo 1

- composition:
  single large dashboard frame + right callout stack
- faux UI:
  top toolbar, left red nav rail, main dashboard blocks
- intended replacement:
  overview screenshot

## 08 Demo 2

- composition:
  top medium desktop frame + bottom mobile shell
- faux UI:
  form rows for contract/invoice, phone shell for mobile usage
- intended replacement:
  contract/invoice screenshot and smartphone screenshot

## 09 Demo 3

- composition:
  2x2 grid
- faux UI:
  each frame has top chrome + small internal skeleton
- intended replacement:
  own publish, other publish, inquiry handling, AI assist

## Shared Rules

- fixed height per slide
- label zone outside the content area
- no giant centered placeholder text
- subtle chrome, not cartoonish
