# Benchmark Prompt Brief

## Shared Direction

- Japanese B2B proposal deck
- red and white corporate presentation
- inspired by the existing Sanko deck but more refined
- clean, serious, premium
- not a flyer, not SNS creative, not consumer marketing

## Board Request

- one contact sheet showing all 13 slides
- each slide clearly separated
- coherent design system across all slides
- readable Japanese headings where possible
- enough detail to judge overall quality, hierarchy, and tone

## Required Tone

- 表紙は余白と重心がきれい
- 目次は単なる箇条書きでなく構造が見える
- 提案の骨子は説得力がある
- 展開イメージは図解として読みやすい
- 今後の進め方は横に流れる工程が分かりやすい
