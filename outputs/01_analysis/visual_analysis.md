# Visual Analysis

## 対象

- 1枚の提案資料表紙スライド
- 種別: 営業提案資料 / プレゼンテーション表紙
- 入力ソース: チャット添付画像を目視解析

## この資料が伝えていること

- 株式会社三幸向けに、基幹管理システムの提案を行う資料である
- 伝えたい主題は「日常業務の効率化」と「将来的な展望」の両立
- 単なる業務システム紹介ではなく、物件・入居者・契約・反響対応を一つの流れで扱う業務基盤の提案である

## 主要メッセージ

- 日常業務の効率化と将来的な展望を見据えた基幹管理システムのご提案

## 重要情報

- 宛先: 株式会社三幸 御中
- ラベル: PROPOSAL
- メインタイトル
- サブタイトル
- 日付
- ブランド: Field X

## 補助情報

- 背景の有機的な赤い図形
- 上部の淡い赤い半円
- 白い大型パネルとロゴ下の小型パネル

## レイアウト特徴

- 16:9のスライド比率を前提にした1ページ構成
- 左寄りに大きな白パネル、その中にタイトル群
- 右側に大きな赤い有機形状
- 左下にも赤い有機形状
- 上中央に淡い赤い半円
- 下部中央右にブランドロゴ

## HTML化で再現すべき要素

- テキストの階層構造
- タイトルのまとまりと余白
- 白パネル主体の構成
- 赤い装飾形状による視線誘導
- Field Xロゴの位置関係

## 再設計してよい要素

- 背景装飾の細かな曲率
- 影の強さとぼかし量
- 厳密なピクセル位置

## 不確かな点

- 元画像の正確なフォントファミリー
- 装飾図形の完全なベジェ形状
- ロゴ下パネルの正確なサイズ感

## 再構築方針

- 画像を背景に敷かず、テキストはHTMLで保持する
- 装飾図形はCSSで再現する
- ロゴのみ画像アセットとして利用する
- PC表示で表紙スライドらしい見栄えを優先しつつ、スマホでは縦積みで読める形に再構成する
