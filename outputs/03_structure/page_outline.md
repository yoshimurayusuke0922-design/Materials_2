# Page Outline

## Page Type

- 単一の提案資料表紙ページ

## Page Goal

- 提案の主題を最初の数秒で把握できるようにする
- 提案先、提案内容、ブランドを明確に示す

## Page Sections

1. 背景装飾レイヤー
2. 宛先ラベル
3. ヒーローパネル
4. 日付
5. ブランドロゴ

## Semantic Tag Policy

- ページ全体: `main`
- 宛先: `header` または `p`
- ヒーロー本体: `section`
- 主要タイトル: `h1`
- 補足説明: `p`
- ロゴ: `footer`

## Responsive Policy

- PCでは16:9のスライドビューを保持
- スマホでは縦積みし、パネルとロゴを自然な順序で再配置
- 装飾図形は一部縮小し、可読性を優先
