# Agenda Slide Builder

## Purpose

提案資料の目次ページを、対外向けの自然な見せ方で構築する。

## Inputs

- 目次項目
- 全体トーン
- デッキの色、余白、タイポグラフィ

## Outputs

- 目次スライドHTML
- 目次スライドCSS
- 目次ページの構成メモ

## Procedure

1. 目次項目を3つ前後に整理する。
2. 行リストでなく、視線誘導しやすいレイアウトを決める。
3. 項目ごとの番号、見出し、補足の有無を整理する。
4. 他ページと整合するトーンで実装する。

## Quality Checks

- 単なる箇条書きに見えないか
- ページ内の重心が偏っていないか
- 目次として次ページを読みたくなる構成か

## Failure Cases

- テキストだけで弱くなる
- デザインを盛りすぎて本文より強くなる

## Notes

- 営業資料なので、社内資料感を出しすぎない。
